ABOUT

Yu-Gi-Oh! Forbidden Memories is a terrible game with a terrible mechanic called "fusions." Fusions allow the player to combine two cards to create a new, hopefully more powerful card.

Naturally, because it is a terrible game, Forbidden Memories does almost nothing to explain which cards fuse together. With more than 700 cards, discovering useful combinations through trial and error becomes tedious very quickly.

THE ORIGINAL PROJECT

This website is a modified fork of the YGO-FM Fusion Calculator originally created by Solumin.

The original calculator was inspired by Giver336's Let's Play of the game on Something Awful. During the playthrough, co-commentator General Yeti suggested the possibility of a program that could find card fusions automatically. Solumin took that idea and created the original project.

THE FORTEFYRE MODIFICATION

This version was modified by Fortefyre because he's bad at the game and wanted the calculator to include cards that are already on the field.

The original calculator supports all five cards in the player's hand. This modified version adds five additional field-card slots and checks possible combinations between cards in the hand and cards currently in play.

Fortefyre did not create the original calculator, fusion database, card database, design, or underlying project. He forked the existing project and expanded its calculator functionality to account for field cards.
